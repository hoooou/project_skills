# 数据库

## Not applicable

该项目是纯文档型 Claude Code Skills 库,不包含数据库或持久化存储层。

## 理由

1. **项目类型**: 这是一个技能定义库,由静态的 Markdown 文件组成
2. **无运行时状态**: 技能本身不运行,不需要保存状态
3. **无用户数据**: 不收集、存储或处理用户数据
4. **文件系统即持久化**: 所有数据以 `.md` 文件形式存在于文件系统

## 相关说明

如果该项目未来需要持久化机制,可能的场景包括:
- 技能使用统计
- 用户偏好设置
- 技能执行历史

但目前这些都不存在,也不在当前项目范围内。

## 目标项目的数据库

需要注意区分:
- **本项目 (skills 库)**: 无数据库
- **目标项目 (被技能操作的项目)**: 可能有数据库

`project-init`、`project-sync` 和 `project-review` 技能会分析和记录**目标项目**的数据库信息,但这不意味着 skills 库本身需要数据库。
